# ออเดอร์และการติดตาม

หลังจากสั่งซื้อแล้ว คุณสามารถติดตามสถานะ รับ credentials ดิจิทัล และดูรายละเอียดทั้งหมดได้จากหน้าออเดอร์

## สถานะออเดอร์

แต่ละออเดอร์จะมีสถานะที่แสดงว่าอยู่ที่ขั้นตอนใดในกระบวนการ:

| สถานะ | ความหมาย |
|---|---|
| รอการชำระเงิน | ออเดอร์ถูกสร้างแล้วแต่ยังไม่ได้รับการชำระเงิน |
| ชำระเงินแล้ว | การชำระเงินได้รับการยืนยัน ออเดอร์กำลังประมวลผล |
| เสร็จสมบูรณ์ | สินค้าส่งมอบหรือจัดส่งแล้ว |
| ยกเลิก | ออเดอร์ถูกยกเลิก |
| คืนเงิน | ออเดอร์ถูกคืนเงินทั้งหมดหรือบางส่วน |

---

## การดูออเดอร์ทั้งหมด

### สำหรับผู้ใช้ที่เข้าสู่ระบบ

1. คลิกชื่อบัญชีในแถบนำทางด้านบน
2. เลือก **ออเดอร์ของฉัน** หรือเข้าไปที่ `/account/orders`
3. ออเดอร์ทั้งหมดของคุณแสดงเรียงตามวันที่ล่าสุด
4. คลิกออเดอร์ใดก็ได้เพื่อดูรายละเอียดทั้งหมด

### สำหรับแขก (Guest)

1. ไปที่ลิงก์ออเดอร์ที่ส่งมาในอีเมลยืนยัน
2. หรือเข้าไปที่ `/orders/[หมายเลขออเดอร์]` โดยตรง
3. หน้ารายละเอียดออเดอร์จะแสดงโดยไม่ต้องเข้าสู่ระบบ

---

## รายละเอียดออเดอร์

หน้ารายละเอียดออเดอร์แสดงข้อมูลทั้งหมดเกี่ยวกับออเดอร์นั้น:

### ข้อมูลออเดอร์

- **หมายเลขออเดอร์** — รหัสอ้างอิงเฉพาะ
- **วันที่สั่งซื้อ** — เมื่อออเดอร์ถูกสร้าง
- **สถานะ** — สถานะปัจจุบันของออเดอร์
- **วิธีการชำระเงิน** — วิธีที่ใช้ชำระเงิน

### รายการสินค้า

แต่ละรายการในออเดอร์แสดง:

- รูปภาพและชื่อสินค้า
- ตัวเลือกที่เลือก (หากมี)
- จำนวน
- ราคาต่อหน่วยและยอดรวม

### ราคา

- ราคาสินค้ารวม
- ค่าจัดส่ง (หากมี)
- ส่วนลดคูปอง (หากใช้)
- **ยอดรวมสุดท้าย**

---

## การรับสินค้าดิจิทัล

สำหรับออเดอร์ที่มีสินค้าดิจิทัล เช่น บัญชีเกม บัญชีสตรีมมิ่ง หรือ license key ออเดอร์จะส่ง credentials ให้กับคุณโดยอัตโนมัติเมื่อชำระเงินสำเร็จ

### การดู Credentials ที่ส่งมอบ

1. เปิดหน้ารายละเอียดออเดอร์
2. หา **ส่วนสินค้าดิจิทัล** หรือ **Credentials ที่ส่งมอบ**
3. ข้อมูลที่แสดงขึ้นอยู่กับประเภทสินค้า ซึ่งอาจรวมถึง:
   - ชื่อผู้ใช้และรหัสผ่าน
   - อีเมลและรหัสผ่าน
   - Serial key หรือ license code
   - ข้อมูลอื่นๆ ที่ร้านค้ากำหนด

### การคัดลอก Credentials

- คลิกไอคอน **คัดลอก** ถัดจากแต่ละรายการเพื่อคัดลอกไปยังคลิปบอร์ด
- บาง credentials อาจมีปุ่ม **คัดลอกทั้งหมด** เพื่อคัดลอกข้อมูลทั้งหมดพร้อมกัน

### การดาวน์โหลด Credentials

หากมีตัวเลือกการดาวน์โหลด:

1. คลิกปุ่ม **ดาวน์โหลด**
2. ไฟล์ข้อความหรือ CSV จะถูกดาวน์โหลดมายังอุปกรณ์ของคุณ

---

## การติดตามการจัดส่ง (สินค้าจริง)

สำหรับออเดอร์สินค้าจริง:

### ก่อนจัดส่ง

ออเดอร์จะแสดงสถานะ **ชำระเงินแล้ว** และรอการเตรียมสินค้า ร้านค้ากำลังดำเนินการกับออเดอร์ของคุณ

### หลังจัดส่ง

เมื่อร้านค้าจัดส่งพัสดุของคุณ:

1. สถานะออเดอร์จะอัปเดตเป็น **จัดส่งแล้ว** หรือ **กำลังจัดส่ง**
2. **หมายเลขติดตาม** จะปรากฏในหน้ารายละเอียดออเดอร์
3. **ชื่อบริษัทขนส่ง** จะแสดงถัดจากหมายเลขติดตาม
4. คลิกหมายเลขติดตามเพื่อไปยังเว็บไซต์ติดตามพัสดุของบริษัทขนส่ง

### เมื่อสินค้าถึงมือ

เมื่อคุณได้รับพัสดุแล้ว สถานะอาจอัปเดตเป็น **เสร็จสมบูรณ์** โดยอัตโนมัติหรือโดยผู้ดูแลระบบ

---

## ออเดอร์บริการ (Service Orders)

สำหรับสินค้าประเภทบริการ (Service Order) กระบวนการแตกต่างออกไปเล็กน้อย:

1. **การสั่งซื้อ** — คุณกรอกข้อมูลที่ร้านค้าต้องการ (ข้อมูลบัญชี รายละเอียดที่ต้องการ ฯลฯ) ในระหว่างชำระเงิน
2. **หลังชำระเงิน** — ทีมงานของร้านค้าตรวจสอบคำขอและดำเนินการบริการ
3. **การตอบกลับ** — ผู้ดูแลระบบอาจส่งข้อความหรืออัปเดตกลับมาในหน้ารายละเอียดออเดอร์
4. **ผลลัพธ์** — เมื่อบริการเสร็จสิ้น ข้อมูลผลลัพธ์จะปรากฏในออเดอร์

---

## การชำระเงินออเดอร์ที่ค้างอยู่

หากออเดอร์ของคุณมีสถานะ **รอการชำระเงิน**:

1. เปิดหน้ารายละเอียดออเดอร์
2. คลิกปุ่ม **ชำระเงินทันที**
3. หรือเข้าไปที่ `/orders/[id]/pay` โดยตรง
4. อัปโหลดสลิปการชำระเงินหรือเลือกวิธีชำระเงินอื่น
5. ส่งการชำระเงิน

ออเดอร์จะไม่ถูกดำเนินการจนกว่าจะได้รับการชำระเงิน
