# การจัดการบัญชีผู้ใช้

เมื่อเข้าสู่ระบบแล้ว คุณสามารถเข้าถึงแดชบอร์ดบัญชีซึ่งรวมฟีเจอร์ทั้งหมดที่เกี่ยวกับโปรไฟล์ ออเดอร์ และบริการของคุณ

## ภาพรวมบัญชี

เข้าไปที่ `/account` หลังจากเข้าสู่ระบบเพื่อดูหน้าภาพรวม หน้านี้แสดง:

- **สถิติสั้นๆ** — จำนวนออเดอร์ทั้งหมด ยอด wallet ปัจจุบัน และสถานะบัญชี
- **ออเดอร์ล่าสุด** — รายการออเดอร์ล่าสุดของคุณพร้อมสถานะ
- **ลิงก์ด่วน** — ปุ่มนำทางไปยังส่วนต่างๆ ของบัญชีเช่น ออเดอร์ wallet และที่อยู่

## ประวัติออเดอร์

เข้าไปที่ `/account/orders` เพื่อดูออเดอร์ทั้งหมดของคุณ

หน้าออเดอร์แสดงออเดอร์ในรูปแบบรายการ แต่ละรายการแสดง:

- หมายเลขออเดอร์
- วันที่สั่งซื้อ
- รายการสินค้า
- ยอดรวม
- สถานะออเดอร์ปัจจุบัน

### การดูรายละเอียดออเดอร์

คลิกออเดอร์ใดก็ได้เพื่อเปิดหน้ารายละเอียด หน้านี้แสดง:

- **สินค้าที่สั่ง** — รายการสินค้าทั้งหมดพร้อมรูปภาพ ชื่อ ตัวเลือก และราคา
- **สถานะการชำระเงิน** — วิธีที่ใช้ชำระและสถานะปัจจุบัน
- **สถานะออเดอร์** — ว่าออเดอร์รอดำเนินการ ชำระแล้ว เสร็จสมบูรณ์ หรือถูกยกเลิก

### การรับสินค้าดิจิทัล

สำหรับออเดอร์ที่มีสินค้าดิจิทัล (เช่น บัญชีเกม หรือ credentials):

1. เปิดหน้ารายละเอียดออเดอร์
2. หา **ส่วน Credentials ที่ส่งมอบ** ซึ่งจะแสดงชื่อผู้ใช้ รหัสผ่าน หรือรายละเอียดอื่นๆ ที่ส่งมอบให้
3. ใช้ปุ่ม **คัดลอก** เพื่อคัดลอก credentials แต่ละรายการไปยังคลิปบอร์ด
4. อาจมีตัวเลือกให้ดาวน์โหลด credentials ในรูปแบบไฟล์ข้อความหรือ CSV

### การติดตามการจัดส่ง

สำหรับออเดอร์ที่มีสินค้าจริง:

1. เปิดหน้ารายละเอียดออเดอร์
2. เมื่อร้านค้าจัดส่งออเดอร์ของคุณ หมายเลขติดตามและชื่อบริษัทขนส่งจะปรากฏ
3. คลิกหมายเลขติดตามหรือลิงก์บริษัทขนส่งเพื่อติดตามสถานะพัสดุบนเว็บไซต์ของบริษัทขนส่ง

## การจัดการที่อยู่

เข้าไปที่ `/account/addresses` เพื่อจัดการที่อยู่จัดส่งที่บันทึกไว้

### การเพิ่มที่อยู่ใหม่

1. คลิก **เพิ่มที่อยู่**
2. กรอกฟอร์มที่อยู่:
   - ชื่อผู้รับ
   - หมายเลขโทรศัพท์
   - บ้านเลขที่และถนน
   - ตำบล อำเภอ จังหวัด และรหัสไปรษณีย์ (โดยใช้ตัวเลือกที่อยู่ไทย)
3. คลิก **บันทึก**
4. ที่อยู่จะปรากฏในรายการที่อยู่ที่บันทึกไว้ และสามารถเลือกได้ในระหว่างการชำระเงิน

### การแก้ไขที่อยู่

1. คลิกปุ่ม **แก้ไข** ถัดจากที่อยู่ที่ต้องการแก้ไข
2. อัปเดตช่องที่ต้องการ
3. คลิก **บันทึก**

### การลบที่อยู่

1. คลิกปุ่ม **ลบ** ถัดจากที่อยู่ที่ต้องการลบ
2. ยืนยันการลบ
3. ที่อยู่จะถูกลบออกจากรายการ

## การสมัครสมาชิก

เข้าไปที่ `/account/subscriptions` เพื่อดูการสมัครสมาชิกที่ใช้งานอยู่

หน้านี้แสดง:

- **การสมัครสมาชิกที่ใช้งานอยู่** — บริการที่คุณสมัครอยู่ในปัจจุบัน
- **วันต่ออายุ** — เมื่อไหร่การสมัครสมาชิกแต่ละรายการจะถูกเรียกเก็บเงินหรือต่ออายุต่อไป
- **รายละเอียดแผน** — สิ่งที่รวมอยู่ในแต่ละการสมัครสมาชิก

คุณสามารถยกเลิกการสมัครสมาชิกได้จากหน้านี้ตามที่ร้านค้าอนุญาต

## การผสานกับ LINE

เข้าไปที่ `/account/line` เพื่อเชื่อมต่อบัญชี LINE ของคุณกับบัญชีร้านค้า

### การเชื่อมต่อบัญชี LINE

1. คลิก **เชื่อมต่อ LINE**
2. คุณจะถูกเปลี่ยนเส้นทางไปยังหน้าการยืนยันตัวตน LINE
3. เข้าสู่ระบบด้วยบัญชี LINE ของคุณ
4. อนุมัติการเชื่อมต่อ
5. คุณจะถูกส่งกลับมายังหน้าบัญชีพร้อมบัญชี LINE ที่เชื่อมต่อแล้ว

เมื่อเชื่อมต่อแล้ว คุณจะได้รับการแจ้งเตือนออเดอร์และการอัปเดตสถานะผ่าน LINE

### การยกเลิกการเชื่อมต่อ LINE

1. บนหน้า LINE ให้คลิก **ยกเลิกการเชื่อมต่อ**
2. ยืนยันการดำเนินการ
3. บัญชี LINE ของคุณจะถูกยกเลิกการเชื่อมต่อและการแจ้งเตือน LINE จะหยุดทำงาน

## API Keys (สำหรับ Reseller)

เข้าไปที่ `/account/api-keys` เพื่อจัดการ API keys สำหรับการผสานระบบภายนอก

ฟีเจอร์นี้มีไว้สำหรับผู้ขายต่อและพาร์ทเนอร์ที่ต้องการเข้าถึง API ของร้านค้า

### การสร้าง API Key

1. คลิก **สร้าง Key ใหม่**
2. ตั้งชื่อ key เพื่อให้จำได้ว่าใช้สำหรับอะไร
3. คลิก **สร้าง**
4. API key จะปรากฏ **ครั้งเดียวเท่านั้น** — คัดลอกและจัดเก็บไว้อย่างปลอดภัยทันที หากคุณปิดหน้าต่างโดยไม่คัดลอก คุณจะต้องสร้าง key ใหม่

### การลบ API Key

1. คลิกปุ่ม **ลบ** ถัดจาก key ที่ต้องการลบ
2. ยืนยันการลบ
3. Key จะถูกเพิกถอนทันที การเรียก API ที่ใช้ key นี้จะล้มเหลว

## ระดับผู้ใช้และราคา VIP

บัญชีของคุณอาจถูกกำหนดระดับโดยผู้ดูแลระบบร้านค้า:

- **ปกติ (Regular)** — ราคามาตรฐาน
- **VIP** — ราคาพิเศษที่ต่ำกว่าในสินค้าที่เลือก
- **พรีเมียม (Premium)** — ราคาพิเศษหรือการเข้าถึงฟีเจอร์ระดับสูงสุด

หากบัญชีของคุณเป็น VIP หรือ Premium หน้ารายละเอียดสินค้าจะแสดงราคาที่ลดลงให้คุณโดยอัตโนมัติ
