# ตะกร้าสินค้าและการชำระเงิน

หลังจากเลือกสินค้าแล้ว ลูกค้าจะดำเนินการผ่านตะกร้าไปยังหน้าชำระเงิน เพื่อกรอกรายละเอียดการจัดส่ง ใช้คูปอง และเลือกวิธีการชำระเงิน

## ตะกร้าสินค้า

### การเพิ่มสินค้า

สินค้าจะถูกเพิ่มลงตะกร้าจากหน้ารายละเอียดสินค้าผ่านกล่องโต้ตอบตัวเลือกการซื้อ เมื่อคุณเลือก **เพิ่มลงตะกร้า** หรือ **ซื้อทันที** สินค้าจะถูกเพิ่มลงตะกร้าและตัวเลขบนไอคอนตะกร้าในแถบนำทางจะเพิ่มขึ้น

### การดูตะกร้า

คลิกไอคอนตะกร้าในแถบนำทาง หรือเข้าไปที่ `/cart` เพื่อเปิดหน้าตะกร้า

หน้าตะกร้าแสดง:

- รายการสินค้าทั้งหมดในตะกร้าปัจจุบัน
- รูปภาพ ชื่อ ตัวเลือกสินค้า ราคาต่อหน่วย และจำนวนของแต่ละสินค้า
- ราคารวมของแต่ละสินค้า
- ราคารวมทั้งหมดของสินค้าที่เลือกที่ด้านล่าง

### การจัดการสินค้าในตะกร้า

บนหน้าตะกร้า คุณสามารถ:

- **เปลี่ยนจำนวน** — ใช้ตัวควบคุมจำนวนถัดจากแต่ละสินค้าเพื่อเพิ่มหรือลดจำนวน ราคารวมจะอัปเดตทันที
- **ลบสินค้า** — คลิกปุ่มลบถัดจากสินค้าเพื่อลบออกจากตะกร้า ราคารวมของตะกร้าจะคำนวณใหม่
- **เลือกสินค้าสำหรับชำระเงิน** — ใช้ช่องทำเครื่องหมายถัดจากแต่ละสินค้าเพื่อเลือกว่าจะรวมสินค้าใดในการชำระเงิน เฉพาะสินค้าที่เลือกเท่านั้นที่จะรวมเมื่อดำเนินการไปยังหน้าชำระเงิน

### ดำเนินการไปยังหน้าชำระเงิน

เมื่อพอใจกับตะกร้าแล้ว คลิกปุ่ม **ชำระเงิน** หรือ **ดำเนินการชำระเงิน** คุณจะถูกพาไปยังหน้าชำระเงินพร้อมสินค้าที่เลือก

---

## การชำระเงิน

หน้าชำระเงินรวบรวมข้อมูลที่จำเป็นเพื่อดำเนินการออเดอร์ให้สมบูรณ์

### การชำระเงินแบบแขก vs. แบบมีบัญชี

- **หากคุณเข้าสู่ระบบแล้ว** รายละเอียดบัญชีของคุณจะถูกกรอกล่วงหน้าในช่องที่เกี่ยวข้อง
- **หากยังไม่ได้เข้าสู่ระบบ** (การชำระเงินแบบแขก) คุณจะต้องให้อีเมลและหมายเลขโทรศัพท์เพื่อให้ร้านส่งการอัปเดตออเดอร์ได้

### ขั้นตอนที่ 1: ข้อมูลติดต่อ (เฉพาะแขก)

หากคุณชำระเงินในฐานะแขก:

1. กรอก **อีเมล** ในช่องที่กำหนด
2. กรอก **หมายเลขโทรศัพท์**
3. ข้อมูลเหล่านี้ใช้สำหรับส่งการยืนยันออเดอร์และแจ้งเตือนการจัดส่ง

### ขั้นตอนที่ 2: ที่อยู่จัดส่ง (เฉพาะสินค้าจริง)

หากตะกร้าของคุณมีสินค้าจริงที่ต้องการจัดส่ง:

1. ฟอร์มกรอกที่อยู่จัดส่งจะปรากฏ
2. กรอก:
   - **ชื่อผู้รับ**
   - **หมายเลขโทรศัพท์**
   - **บ้านเลขที่และถนน**
   - **ตำบล/แขวง**
   - **อำเภอ/เขต**
   - **จังหวัด**
   - **รหัสไปรษณีย์**
3. ฟอร์มใช้ตัวเลือกที่อยู่ไทยที่จะแสดงตัวเลือกให้แคบลงขณะที่คุณพิมพ์ เริ่มต้นด้วยจังหวัดหรือรหัสไปรษณีย์ และช่องอื่นๆ จะกรอกค่าที่ตรงกันโดยอัตโนมัติ
4. หากคุณมีที่อยู่ที่บันทึกไว้ในบัญชี จะมีเมนูแบบเลื่อนลงให้เลือกที่อยู่ที่บันทึกไว้และกรอกข้อมูลทุกช่องโดยอัตโนมัติ

### ขั้นตอนที่ 3: วิธีการจัดส่ง (เฉพาะสินค้าจริง)

หลังจากกรอกที่อยู่จัดส่งแล้ว:

1. บริษัทขนส่งที่มีให้บริการจะแสดงพร้อมราคาโดยประมาณ
2. คลิกบริษัทขนส่งที่ต้องการเพื่อเลือกวิธีการจัดส่ง
3. ค่าจัดส่งจะถูกรวมเข้ากับยอดรวมของออเดอร์

### ขั้นตอนที่ 4: รหัสคูปอง (ไม่บังคับ)

หากคุณมีคูปองส่วนลด:

1. ค้นหาช่อง **รหัสคูปอง** ในฟอร์มชำระเงิน
2. กรอกรหัสคูปองของคุณ
3. คลิก **ใช้คูปอง**
4. หากรหัสถูกต้อง ส่วนลดจะแสดงและยอดรวมจะลดลง
5. หากรหัสไม่ถูกต้องหรือหมดอายุ จะมีข้อความแสดงข้อผิดพลาดปรากฏ

สามารถใช้คูปองได้เพียงหนึ่งใบต่อออเดอร์

### ขั้นตอนที่ 5: วิธีการชำระเงิน

เลือกวิธีที่คุณต้องการชำระเงิน:

#### Wallet / ยอดเครดิต

หากคุณมียอด wallet เพียงพอ:

1. ตัวเลือก wallet จะแสดงยอดคงเหลือปัจจุบันของคุณ
2. เลือก **ชำระด้วย Wallet**
3. ยอดเงินจะถูกหักจากยอดคงเหลือเมื่อสั่งซื้อ
4. ออเดอร์จะได้รับการดำเนินการทันที

#### สลิปการชำระเงิน (โอนเงินผ่านธนาคาร)

สำหรับการชำระเงินผ่านการโอนเงินธนาคาร:

1. เลือก **สลิปการชำระเงิน**
2. QR Code (PromptPay) และรายละเอียดบัญชีธนาคารจะแสดง
3. โอนเงินตามจำนวนออเดอร์จากแอปธนาคารของคุณ
4. อัปโหลดภาพหน้าจอหรือรูปถ่ายของการยืนยันการโอนเป็นสลิปการชำระเงิน
5. คลิก **ส่งออเดอร์**
6. ออเดอร์ของคุณถูกสร้างด้วยสถานะ **รอการชำระเงิน**
7. ผู้ดูแลระบบร้านตรวจสอบสลิปของคุณและอนุมัติหรือปฏิเสธ
8. เมื่ออนุมัติแล้ว สถานะออเดอร์จะเปลี่ยนเป็น **ชำระเงินแล้ว** และสินค้าของคุณจะได้รับการประมวลผล

#### Payso Gateway

1. เลือกตัวเลือกการชำระเงิน Payso
2. คลิก **ดำเนินการชำระเงิน**
3. คุณจะถูกเปลี่ยนเส้นทางไปยังหน้าชำระเงิน Payso
4. ชำระเงินโดยใช้วิธีที่มีในหน้านั้น
5. หลังจากชำระเงินสำเร็จ คุณจะถูกเปลี่ยนเส้นทางกลับมายังหน้าร้านและออเดอร์ของคุณได้รับการยืนยัน

#### Beam Gateway

1. เลือกตัวเลือกการชำระเงิน Beam
2. คลิก **ดำเนินการชำระเงิน**
3. คุณจะถูกเปลี่ยนเส้นทางไปยังหน้าชำระเงิน Beam
4. ดำเนินการชำระเงินบน Beam ให้เสร็จสมบูรณ์
5. หลังจากยืนยันการชำระเงิน คุณจะถูกเปลี่ยนเส้นทางกลับมายังหน้าร้านพร้อมออเดอร์ที่สมบูรณ์

### ขั้นตอนที่ 6: การยืนยันออเดอร์

หลังจากชำระเงินเสร็จสมบูรณ์:

1. คุณจะถูกพาไปยังหน้ายืนยันออเดอร์หรือหน้ารายละเอียดออเดอร์
2. อีเมลยืนยันจะถูกส่งไปยังอีเมลของคุณ
3. สำหรับ **สินค้าดิจิทัล** credentials (ชื่อผู้ใช้ รหัสผ่าน หรือ license key) จะแสดงบนหน้ารายละเอียดออเดอร์ทันที
4. สำหรับ **สินค้าจริง** ออเดอร์จะถูกจัดคิวสำหรับการจัดส่ง และคุณจะได้รับหมายเลขติดตามเมื่อสินค้าถูกจัดส่งออกไป

## หน้ารายละเอียดออเดอร์ (หลังชำระเงิน)

คุณสามารถดูสถานะออเดอร์ได้ตลอดเวลาโดย:

- เข้าไปที่ `/account/orders` หากเข้าสู่ระบบแล้ว จากนั้นคลิกออเดอร์
- ใช้ลิงก์ออเดอร์จากอีเมลยืนยัน (ใช้ได้สำหรับแขกด้วย) ที่ `/orders/[id]`

หน้ารายละเอียดออเดอร์แสดง:

- หมายเลขออเดอร์และวันที่
- สินค้าที่สั่งพร้อมจำนวนและราคา
- วิธีการชำระเงินและสถานะ
- สำหรับสินค้าดิจิทัล: credentials ที่ส่งมอบ
- สำหรับสินค้าจริง: หมายเลขติดตามและบริษัทขนส่งเมื่อจัดส่งแล้ว
- ปุ่มชำระเงินหากการชำระเงินยังรอดำเนินการ

---

## ออเดอร์ที่รอการชำระเงิน

หากคุณสั่งซื้อด้วยสลิปการชำระเงินแต่ยังไม่ได้ชำระ:

1. เข้าไปที่ `/orders/[id]/pay` หรือคลิกลิงก์ชำระเงินในหน้ารายละเอียดออเดอร์
2. อินเทอร์เฟซอัปโหลดสลิปการชำระเงินจะแสดงขึ้นอีกครั้ง
3. อัปโหลดสลิปของคุณและส่งใหม่

หรืออีกวิธีหนึ่ง เข้าไปที่หน้ารายละเอียดออเดอร์และคลิกปุ่ม **ชำระเงินทันที**
