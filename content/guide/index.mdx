# เอกสาร Seluna Cloud

ยินดีต้อนรับสู่เอกสารอ้างอิงของ Seluna Cloud เลือกส่วนที่ตรงกับบทบาทของคุณด้านล่าง

---

## หน้าร้าน (Storefront) — สำหรับลูกค้า

| หัวข้อ | คำอธิบาย |
|---|---|
| [การยืนยันตัวตน](./storefront/auth.mdx) | สมัครสมาชิก เข้าสู่ระบบ ลืมรหัสผ่าน และออกจากระบบ |
| [การเรียกดูสินค้า](./storefront/browsing.mdx) | หน้าแรก หน้าร้าน การค้นหา และหน้ารายละเอียดสินค้า |
| [ตะกร้าและการชำระเงิน](./storefront/cart-checkout.mdx) | จัดการตะกร้า ขั้นตอนชำระเงิน และวิธีการชำระเงิน |
| [การจัดการบัญชี](./storefront/account.mdx) | บัญชีผู้ใช้ ออเดอร์ ที่อยู่ API keys และการเชื่อมต่อ LINE |
| [Wallet และการเติมเงิน](./storefront/wallet.mdx) | ยอดเครดิต ประวัติธุรกรรม และวิธีเติมเงิน |
| [ออเดอร์และการติดตาม](./storefront/orders.mdx) | ดูออเดอร์ รับ credentials ดิจิทัล และติดตามการจัดส่ง |
| [บล็อกและการติดต่อ](./storefront/blog-contact.mdx) | อ่านบล็อก ส่งข้อความติดต่อ และแจ้งปัญหา |

---

## แดชบอร์ดผู้ดูแลระบบ (Admin) — สำหรับผู้ดูแลร้านค้า

| หัวข้อ | คำอธิบาย |
|---|---|
| [แดชบอร์ดหลัก](./admin/dashboard.mdx) | ภาพรวมตัวชี้วัด การแจ้งเตือน และแถบนำทางหลังบ้าน |
| [สินค้าและหมวดหมู่](./admin/products.mdx) | สร้าง แก้ไข และจัดการสินค้าและหมวดหมู่ |
| [สต็อกดิจิทัล](./admin/stock.mdx) | จัดการ credentials นำเข้า CSV และติดตามสต็อก |
| [ออเดอร์](./admin/orders.mdx) | ดู อนุมัติ อัปเดต และคืนเงินออเดอร์ |
| [คูปองและเติมเงิน](./admin/coupons-topups.mdx) | สร้างคูปอง อนุมัติการเติมเงิน และจัดการคูปองเติมเงิน |
| [ผู้ใช้และ Reseller](./admin/users.mdx) | จัดการบัญชีลูกค้า ระดับผู้ใช้ และบัญชี reseller |
| [การตั้งค่าเว็บไซต์](./admin/settings.mdx) | ปรับแต่งแบรนด์ ธีม หน้าแรก เมนู การชำระเงิน และอื่นๆ |
| [การจัดส่ง](./admin/shipping.mdx) | โซนจัดส่ง พัสดุ และการผสานกับบริษัทขนส่ง |
| [บล็อกและอีเมล](./admin/blog-email.mdx) | สร้างบทความ ปรับแต่งเทมเพลตอีเมล และดูประวัติการส่ง |
| [สมัครสมาชิกและ LINE](./admin/subscriptions-line.mdx) | จัดการการสมัครสมาชิก ผู้ใช้ LINE และฟีเจอร์อื่นๆ |

## Portal — สำหรับเจ้าของร้านค้า

| หัวข้อ | คำอธิบาย |
|---|---|
| [ภาพรวม Portal](./portal/overview.mdx) | วัตถุประสงค์ของ Portal การเข้าสู่ระบบ และการนำทาง |
| [การจัดการ Sites](./portal/sites.mdx) | สร้าง ตั้งค่า และจัดการ site ของร้านค้า |
| [การดูแลระบบแพลตฟอร์ม](./portal/platform-admin.mdx) | การจัดการระดับแพลตฟอร์มสำหรับ superadmin |

---

## เริ่มต้นอย่างรวดเร็ว

**ฉันเป็นลูกค้าใหม่** → เริ่มจาก [การยืนยันตัวตน](./storefront/auth.mdx) แล้วอ่าน [การเรียกดูสินค้า](./storefront/browsing.mdx)

**ฉันต้องการสั่งซื้อ** → ดู [ตะกร้าและการชำระเงิน](./storefront/cart-checkout.mdx)

**ฉันเป็นผู้ดูแลระบบร้านใหม่** → เริ่มจาก [แดชบอร์ดหลัก](./admin/dashboard.mdx) แล้วไปที่ [สินค้า](./admin/products.mdx) และ [การตั้งค่า](./admin/settings.mdx)

**ฉันต้องการเปิดร้านใหม่** → เริ่มจาก [ภาพรวม Portal](./portal/overview.mdx) และ [การจัดการ Sites](./portal/sites.mdx)
