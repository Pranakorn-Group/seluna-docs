# การจัดการผู้ใช้และ Reseller

## การจัดการลูกค้า

### การดูรายการลูกค้า

ไปที่ **แดชบอร์ด → ลูกค้า** (`/backend/users`) เพื่อดูบัญชีลูกค้าทั้งหมดในร้านค้า

รายการแสดง:
- ชื่อและอีเมลลูกค้า
- วันที่ลงทะเบียน
- ระดับผู้ใช้ (ปกติ, VIP, พรีเมียม)
- ยอด Wallet
- จำนวนออเดอร์

### การดูโปรไฟล์ลูกค้า

คลิกลูกค้าเพื่อดูโปรไฟล์ละเอียด:
- ข้อมูลส่วนตัว
- ประวัติออเดอร์ทั้งหมด
- ยอด wallet และประวัติธุรกรรม
- ที่อยู่ที่บันทึกไว้

### การเปลี่ยนระดับผู้ใช้

ระดับผู้ใช้ส่งผลต่อราคาที่ลูกค้าเห็น:

1. เปิดโปรไฟล์ลูกค้า
2. หาส่วน **ระดับผู้ใช้ (User Tier)**
3. เลือกระดับที่ต้องการ:
   - **REGULAR** — ราคาปกติ
   - **VIP** — เห็นราคา VIP ที่ต่ำกว่าในสินค้าที่กำหนด
   - **PREMIUM** — ระดับสูงสุด อาจมีสิทธิพิเศษเพิ่มเติม
4. บันทึกการเปลี่ยนแปลง
5. ลูกค้าจะเห็นราคาที่อัปเดตทันทีในครั้งถัดไปที่เข้าใช้งาน

### การปรับยอด Wallet

หากต้องการเพิ่มหรือลดเครดิตในบัญชีลูกค้าด้วยตนเอง:

1. เปิดโปรไฟล์ลูกค้า
2. หาส่วน **Wallet**
3. คลิก **ปรับยอด** หรือ **เพิ่มเครดิต**
4. กรอกจำนวนเงิน (บวกเพื่อเพิ่ม ลบเพื่อหัก)
5. กรอกหมายเหตุสำหรับการปรับยอด
6. ยืนยัน
7. ธุรกรรมจะปรากฏในประวัติ wallet ของลูกค้า

### การระงับบัญชีลูกค้า

หากต้องการระงับการเข้าถึงของลูกค้า:

1. เปิดโปรไฟล์ลูกค้า
2. คลิก **ระงับบัญชี**
3. ยืนยัน
4. ลูกค้าจะไม่สามารถเข้าสู่ระบบได้จนกว่าจะยกเลิกการระงับ

---

## การจัดการ Reseller

Reseller คือบัญชีพิเศษสำหรับพาร์ทเนอร์ที่ขายสินค้าต่อหรือผสานระบบกับ API ของร้านค้า

### การดูรายการ Reseller

ไปที่ **แดชบอร์ด → ผู้ขายต่อ (Resellers)** (`/backend/resellers`)

รายการแสดง:
- ชื่อและอีเมล reseller
- สถานะบัญชี
- การใช้งาน API
- วันที่เข้าร่วม

### การสร้างบัญชี Reseller

1. คลิก **เพิ่ม Reseller**
2. กรอกข้อมูล:
   - อีเมล
   - ชื่อ
   - ระดับสิทธิ์การเข้าถึง
3. คลิก **บันทึก**
4. บัญชี reseller ถูกสร้างและสามารถสร้าง API key ได้

### การจัดการสิทธิ์ Reseller

ผู้ดูแลระบบสามารถกำหนดว่า reseller เข้าถึงสินค้า ราคา และฟีเจอร์ใดได้บ้าง ขึ้นอยู่กับข้อตกลงกับแต่ละพาร์ทเนอร์

### การระงับ Reseller

1. เปิดโปรไฟล์ reseller
2. คลิก **ระงับ**
3. ยืนยัน
4. API key ทั้งหมดของ reseller นี้จะหยุดทำงานทันที

## ข้อความติดต่อ

เมื่อลูกค้าส่งข้อความผ่านหน้าติดต่อ ข้อความจะปรากฏที่นี่

### การดูข้อความ

ไปที่ **แดชบอร์ด → ข้อความติดต่อ** (`/backend/contact-messages`)

รายการแสดง:
- ชื่อผู้ส่ง
- อีเมล
- หัวข้อหรือบทสรุปข้อความ
- วันที่ส่ง
- สถานะ (ยังไม่อ่าน/อ่านแล้ว)

### การตอบกลับข้อความ

ข้อความเหล่านี้มีไว้สำหรับทีมงานอ่านและตอบกลับผ่านทางอีเมลโดยตรง ไม่มีระบบตอบกลับอัตโนมัติในหน้านี้ ต้องติดต่อลูกค้ากลับผ่านอีเมลที่ลูกค้าให้ไว้
