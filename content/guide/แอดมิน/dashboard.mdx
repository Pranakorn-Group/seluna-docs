# แดชบอร์ดผู้ดูแลระบบ

แดชบอร์ดผู้ดูแลระบบ (`/backend`) คือศูนย์กลางในการบริหารร้านค้าทั้งหมด เข้าถึงได้เฉพาะผู้ใช้ที่มีบทบาทเป็น ADMIN เท่านั้น

## การเข้าถึงแดชบอร์ด

1. เข้าสู่ระบบด้วยบัญชีผู้ดูแลระบบ
2. เข้าไปที่ `/backend` หรือคลิกลิงก์แดชบอร์ดจากเมนูบัญชี
3. แดชบอร์ดหลักจะแสดงขึ้น

หากคุณไม่มีสิทธิ์ผู้ดูแลระบบ ระบบจะเปลี่ยนเส้นทางคุณกลับไปยังหน้าแรกของหน้าร้าน

---

## ภาพรวมแดชบอร์ด

หน้าแดชบอร์ดหลักแสดงตัวชี้วัดประสิทธิภาพร้านค้าแบบเรียลไทม์

### ตัวชี้วัดหลัก (Key Metrics)

ที่ด้านบนของแดชบอร์ด การ์ดสรุปแสดง:

- **รายได้ทั้งหมด** — ยอดขายรวมสำหรับช่วงเวลาที่เลือก
- **จำนวนออเดอร์** — ออเดอร์ทั้งหมดที่สร้างขึ้น
- **ลูกค้าใหม่** — ผู้ใช้ที่ลงทะเบียนใหม่
- **สินค้าขายดี** — สินค้าที่มียอดขายสูงสุด

### กราฟและแผนภูมิ

- **กราฟรายได้** — แสดงแนวโน้มรายได้ตามเวลา (รายวัน รายสัปดาห์ หรือรายเดือน)
- **กราฟออเดอร์** — จำนวนออเดอร์แต่ละวัน

### การแจ้งเตือน

ส่วนการแจ้งเตือนแสดงรายการที่ต้องดำเนินการทันที เช่น:

- ออเดอร์ที่รอการอนุมัติ
- คำขอเติมเงินที่รอตรวจสอบ
- สต็อกสินค้าที่ใกล้หมด
- สลิปการชำระเงินที่รอตรวจสอบ

### ออเดอร์ล่าสุด

รายการออเดอร์ล่าสุด 10-20 รายการแสดงพร้อม:

- หมายเลขออเดอร์
- ชื่อลูกค้า
- สินค้า
- ยอดรวม
- สถานะ
- ปุ่มดูรายละเอียด

## แถบนำทางด้านข้าง (Sidebar)

แถบนำทางด้านข้างแสดงรายการเมนูทั้งหมด จัดกลุ่มตามหมวดหมู่:

| หมวดหมู่ | รายการเมนู |
|---|---|
| ภาพรวม | แดชบอร์ด |
| การตั้งค่า | ตั้งค่าเว็บไซต์ (14 แท็บ) |
| แคตาล็อก | สินค้า, หมวดหมู่, สต็อก |
| การขาย | ออเดอร์, คูปอง, เติมเงิน, คูปองเติมเงิน |
| การสมัครสมาชิก | การสมัครสมาชิก |
| เนื้อหา | บล็อก, เทมเพลตอีเมล, ประวัติอีเมล |
| การจัดส่ง | การจัดส่ง, คลังสินค้าการจัดส่ง |
| ผู้ใช้ | ลูกค้า, ผู้ขายต่อ |
| บริการ | ออเดอร์บริการ, ข้อความติดต่อ |
| การผสาน LINE | ผู้ใช้ LINE, ออเดอร์ LINE, ธุรกรรม LINE |
| อื่นๆ | Gamepass, รายงานร้านค้า, ร้านค้าต้นทาง, การซื้อต้นทาง |

คลิกรายการใดก็ได้เพื่อนำทางไปยังส่วนนั้น

## การใช้งานทั่วไปในแดชบอร์ด

### การดูออเดอร์ที่ต้องดำเนินการ

1. ที่แดชบอร์ดหลัก มองหาส่วน **ต้องดำเนินการ** หรือ **ออเดอร์รอดำเนินการ**
2. คลิกออเดอร์เพื่อดูรายละเอียด
3. หรือไปที่ **ออเดอร์** ในแถบด้านข้างเพื่อดูรายการทั้งหมด

### การอนุมัติคำขอเติมเงิน

1. ที่แดชบอร์ด มองหาการแจ้งเตือน **คำขอเติมเงินรอดำเนินการ**
2. คลิกเพื่อไปยังหน้าเติมเงิน
3. ตรวจสอบสลิปและอนุมัติหรือปฏิเสธ

### การตรวจสอบสต็อกต่ำ

1. แดชบอร์ดแสดงการแจ้งเตือนเมื่อสต็อกต่ำกว่าเกณฑ์
2. คลิกการแจ้งเตือนเพื่อไปยังหน้าสต็อก
3. เพิ่มสินค้าเข้าสต็อกตามความจำเป็น
