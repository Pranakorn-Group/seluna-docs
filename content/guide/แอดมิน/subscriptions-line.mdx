# การสมัครสมาชิกและการผสาน LINE

## การจัดการการสมัครสมาชิก

การสมัครสมาชิก (Subscriptions) คือบริการแบบรายงวดที่ลูกค้าจ่ายซ้ำๆ เช่น รายเดือนหรือรายปี

### การดูการสมัครสมาชิกทั้งหมด

ไปที่ **แดชบอร์ด → การสมัครสมาชิก** (`/backend/subscriptions`)

รายการแสดง:
- ชื่อลูกค้า
- แผนที่สมัคร
- วันที่เริ่มต้น
- วันต่ออายุต่อไป
- สถานะ (ใช้งานอยู่, หมดอายุ, ยกเลิก)

### การดูรายละเอียดการสมัครสมาชิก

คลิกการสมัครสมาชิกเพื่อดู:
- ประวัติการชำระเงินทั้งหมด
- สถานะปัจจุบัน
- วันหมดอายุ

### การยกเลิกการสมัครสมาชิก

1. เปิดการสมัครสมาชิกที่ต้องการยกเลิก
2. คลิก **ยกเลิกการสมัครสมาชิก**
3. ยืนยัน
4. ลูกค้าจะได้รับการแจ้งเตือนว่าการสมัครสมาชิกถูกยกเลิก

## การผสาน LINE

ระบบผสานกับ LINE Bot เพื่อส่งการแจ้งเตือนและให้ลูกค้าสั่งซื้อผ่าน LINE ได้

### ผู้ใช้ LINE

ไปที่ **แดชบอร์ด → ผู้ใช้ LINE** (`/backend/line-users`)

หน้านี้แสดงรายการลูกค้าที่เชื่อมต่อบัญชี LINE กับบัญชีร้านค้า:
- ชื่อ LINE
- อีเมลบัญชีร้านค้า
- วันที่เชื่อมต่อ
- สถานะการเชื่อมต่อ

### ออเดอร์ LINE

ไปที่ **แดชบอร์ด → ออเดอร์ LINE** (`/backend/line-orders`)

หน้านี้แสดงออเดอร์ที่มาจากช่องทาง LINE โดยเฉพาะ ข้อมูลเหมือนกับออเดอร์ปกติ แต่กรองเฉพาะที่มาจาก LINE

### ธุรกรรม LINE

ไปที่ **แดชบอร์ด → ธุรกรรม LINE** (`/backend/line-transactions`)

หน้านี้บันทึกธุรกรรมทั้งหมดที่เกิดผ่านช่องทาง LINE รวมถึง:
- การชำระเงิน
- การเติมเงิน
- การโอนเครดิต

## Gamepass

ไปที่ **แดชบอร์ด → Gamepass** (`/backend/gamepass`) เพื่อจัดการบัญชีและ license เกม

หน้านี้ใช้สำหรับร้านค้าที่ขายสินค้าเกี่ยวกับเกม:
- จัดการบัญชีเกมที่จำหน่าย
- ติดตามสถานะการใช้งาน license
- ดูประวัติการแลกใช้

---

## รายงานร้านค้า

ไปที่ **แดชบอร์ด → รายงานร้านค้า** (`/backend/shop-reports`)

เมื่อลูกค้าส่งรายงานร้านค้าผ่านปุ่มในส่วนท้ายเว็บ รายงานเหล่านั้นจะปรากฏที่นี่:
- วันที่รายงาน
- ประเภทปัญหา
- รายละเอียดที่ลูกค้าระบุ
- สถานะการตรวจสอบ

ผู้ดูแลระบบสามารถตรวจสอบและดำเนินการตามรายงานเหล่านี้ได้

---

## ร้านค้าต้นทาง (Source Shops)

ไปที่ **แดชบอร์ด → ร้านค้าต้นทาง** (`/backend/source-shops`)

หน้านี้จัดการแหล่งที่มาของสินค้าหรือบัญชีที่จัดซื้อมาจากซัพพลายเออร์หรือผู้ขายรายอื่น

### การซื้อต้นทาง (Source Purchases)

ไปที่ **แดชบอร์ด → การซื้อต้นทาง** (`/backend/source-purchases`)

บันทึกการซื้อสินค้าจากซัพพลายเออร์ต้นทาง เพื่อติดตามต้นทุนสินค้า
