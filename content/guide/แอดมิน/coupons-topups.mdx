# คูปองและการเติมเงิน

## คูปองส่วนลด

คูปองช่วยให้ลูกค้าได้รับส่วนลดเมื่อสั่งซื้อสินค้า

### การดูคูปองทั้งหมด

ไปที่ **แดชบอร์ด → คูปอง** (`/backend/coupons`) เพื่อดูรายการคูปองทั้งหมด

รายการแสดง:
- รหัสคูปอง
- ประเภทส่วนลด (เปอร์เซ็นต์หรือจำนวนเงิน)
- จำนวนส่วนลด
- จำนวนครั้งที่ใช้
- สถานะ (ใช้งานได้/หมดอายุ)

### การสร้างคูปองใหม่

1. คลิก **สร้างคูปอง**
2. กรอกข้อมูล:
   - **รหัสคูปอง** — รหัสที่ลูกค้าจะใช้ (เช่น SAVE10, WELCOME20)
   - **ประเภทส่วนลด**:
     - **เปอร์เซ็นต์** — ลดตามเปอร์เซ็นต์ของยอดรวม เช่น 10% off
     - **จำนวนเงิน** — ลดจำนวนเงินคงที่ เช่น ลด 50 บาท
   - **จำนวนส่วนลด** — ตัวเลขส่วนลด (เช่น 10 สำหรับ 10% หรือ 50 สำหรับ 50 บาท)
   - **จำนวนการใช้สูงสุด** — จำกัดจำนวนครั้งที่ลูกค้าทั้งหมดสามารถใช้คูปองนี้ได้ (ว่างเปล่า = ไม่จำกัด)
   - **วันหมดอายุ** — วันที่คูปองหยุดใช้งาน (ว่างเปล่า = ไม่มีวันหมดอายุ)
3. คลิก **บันทึก**
4. คูปองพร้อมใช้งานสำหรับลูกค้าทันที

### การแก้ไขคูปอง

1. คลิกปุ่ม **แก้ไข** ถัดจากคูปอง
2. อัปเดตข้อมูลที่ต้องการ
3. คลิก **บันทึก**

### การลบหรือปิดใช้งานคูปอง

1. คลิกปุ่ม **ลบ** หรือ **ปิดใช้งาน** ถัดจากคูปอง
2. ยืนยันการดำเนินการ
3. คูปองที่ปิดใช้งานจะไม่สามารถใช้ในการสั่งซื้อใหม่ได้

## การตรวจสอบคำขอเติมเงิน

เมื่อลูกค้าส่งคำขอเติมเงินด้วยสลิปการโอนเงิน ผู้ดูแลระบบต้องตรวจสอบและอนุมัติ

### การดูคำขอเติมเงิน

ไปที่ **แดชบอร์ด → เติมเงิน** (`/backend/topups`) เพื่อดูคำขอทั้งหมด

รายการแสดง:
- ชื่อลูกค้า
- จำนวนเงินที่ขอเติม
- วันที่ส่งคำขอ
- สถานะ (รอดำเนินการ, อนุมัติ, ปฏิเสธ)
- ปุ่มดูสลิปและดำเนินการ

### การอนุมัติคำขอเติมเงิน

1. คลิกคำขอที่มีสถานะ **รอดำเนินการ**
2. ดูรูปสลิปการโอนเงินที่แนบมา
3. ตรวจสอบ:
   - ยอดเงินที่โอนตรงกับจำนวนที่ขอเติม
   - วันที่และเวลาสมเหตุสมผล
   - บัญชีปลายทางเป็นของร้านค้า
4. หากถูกต้อง คลิก **อนุมัติ**
5. เครดิตจะถูกเพิ่มเข้า wallet ของลูกค้าทันที
6. ลูกค้าได้รับการแจ้งเตือนว่าการเติมเงินสำเร็จ

### การปฏิเสธคำขอเติมเงิน

1. เปิดคำขอที่ต้องการปฏิเสธ
2. คลิก **ปฏิเสธ**
3. กรอกเหตุผลการปฏิเสธ (เช่น สลิปไม่ชัดเจน ยอดไม่ตรง)
4. ยืนยัน
5. ลูกค้าได้รับการแจ้งเตือนพร้อมเหตุผลการปฏิเสธ

## คูปองเติมเงิน (Topup Coupons)

คูปองเติมเงินคือรหัสพิเศษที่ลูกค้าสามารถใช้เพื่อเพิ่มเครดิตเข้า wallet โดยไม่ต้องโอนเงิน

### การดูคูปองเติมเงิน

ไปที่ **แดชบอร์ด → คูปองเติมเงิน** (`/backend/topup-coupons`)

### การสร้างคูปองเติมเงิน

1. คลิก **สร้างคูปองเติมเงิน**
2. กรอก:
   - **รหัสคูปอง** — รหัสที่ลูกค้าจะใช้เพื่อรับเครดิต
   - **จำนวนเครดิต** — จำนวนเครดิตที่จะเพิ่มเข้า wallet
   - **จำนวนการใช้สูงสุด** — ว่าคูปองนี้ใช้ได้กี่ครั้ง
   - **วันหมดอายุ** — ไม่บังคับ
3. คลิก **บันทึก**
4. แจกจ่ายรหัสคูปองให้ลูกค้าผ่านช่องทางต่างๆ (อีเมล LINE ฯลฯ)

### การใช้งานจาก Admin

คูปองเติมเงินมีประโยชน์สำหรับ:
- โปรโมชันพิเศษ (เช่น "เติม 100 บาท ได้ 120 บาท")
- ชดเชยปัญหาที่เกิดขึ้นกับลูกค้า
- รางวัลสำหรับลูกค้าที่ใช้บริการบ่อย
- สินค้าพิเศษที่จ่ายด้วยรหัส
