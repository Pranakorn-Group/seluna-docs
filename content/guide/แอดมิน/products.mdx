# การจัดการสินค้าและหมวดหมู่

## หมวดหมู่สินค้า

ก่อนสร้างสินค้า ควรตั้งค่าหมวดหมู่ก่อนเพื่อให้สินค้าถูกจัดกลุ่มอย่างถูกต้อง

### การดูหมวดหมู่

ไปที่ **แดชบอร์ด → หมวดหมู่** (`/backend/categories`) เพื่อดูรายการหมวดหมู่ทั้งหมด

รายการแสดง:
- ชื่อหมวดหมู่
- ประเภทหมวดหมู่
- จำนวนสินค้าในหมวดหมู่
- สถานะ (เปิดใช้งาน/ปิดใช้งาน)

### การสร้างหมวดหมู่ใหม่

1. คลิก **เพิ่มหมวดหมู่**
2. กรอก:
   - **ชื่อหมวดหมู่** — ชื่อที่แสดงบนหน้าร้าน
   - **ประเภท** — ประเภทสินค้าในหมวดหมู่นี้ (ดิจิทัล, จัดส่ง, บริการ ฯลฯ)
3. คลิก **บันทึก**
4. หมวดหมู่ใหม่ปรากฏในรายการและพร้อมใช้งานเมื่อสร้างสินค้า

### การแก้ไขหมวดหมู่

1. คลิกปุ่ม **แก้ไข** ถัดจากหมวดหมู่
2. อัปเดตช่องที่ต้องการ
3. คลิก **บันทึก**

### การลบหมวดหมู่

1. คลิกปุ่ม **ลบ** ถัดจากหมวดหมู่
2. ยืนยันการลบ

หมายเหตุ: ไม่สามารถลบหมวดหมู่ที่มีสินค้าอยู่ได้ ต้องย้ายหรือลบสินค้าในหมวดหมู่นั้นก่อน

## สินค้า

### การดูสินค้าทั้งหมด

ไปที่ **แดชบอร์ด → สินค้า** (`/backend/products`) เพื่อดูรายการสินค้าทั้งหมด

รายการแสดง:
- รูปภาพสินค้า
- ชื่อสินค้า
- หมวดหมู่
- ราคา
- ประเภทสินค้า
- สถานะสต็อก
- ปุ่มแก้ไขและลบ

### การสร้างสินค้าใหม่

1. คลิก **เพิ่มสินค้า**
2. กรอกข้อมูลสินค้า:

#### ข้อมูลพื้นฐาน

- **ชื่อสินค้า** — ชื่อที่แสดงบนหน้าร้าน
- **คำอธิบาย** — คำอธิบายละเอียดของสินค้า
- **หมวดหมู่** — เลือกจากหมวดหมู่ที่มีอยู่
- **รูปภาพสินค้า** — อัปโหลดรูปภาพหลักของสินค้า คลิกพื้นที่อัปโหลดและเลือกไฟล์รูปภาพ

#### ประเภทสินค้า

เลือกประเภทสินค้าซึ่งกำหนดวิธีการส่งมอบ:

| ประเภท | คำอธิบาย |
|---|---|
| DIGITAL_ACCOUNT | สินค้าดิจิทัลที่ส่งมอบ credentials อัตโนมัติ |
| DIGITAL_ACCOUNT_API | สินค้าดิจิทัลที่ดึงข้อมูลผ่าน API ภายนอก |
| SERVICE_ORDER | บริการที่ต้องมีข้อมูลจากลูกค้าและดำเนินการโดยทีมงาน |
| SERVICE_ORDER_API | บริการที่ส่งผ่าน API อัตโนมัติ |
| SHIPPING | สินค้าจริงที่จัดส่งทางไปรษณีย์ |

#### ราคา

- **ราคาปกติ** — ราคาสำหรับลูกค้าทั่วไป
- **ราคา VIP** — ราคาพิเศษสำหรับลูกค้าระดับ VIP (ไม่บังคับ)
- **ราคาเปรียบเทียบ** — ราคาเดิมที่แสดงเพื่อเปรียบเทียบ (ไม่บังคับ สำหรับแสดงว่าลดราคา)
- **คอยน์ที่ได้รับ** — จำนวนคอยน์ที่ลูกค้าได้รับเมื่อซื้อสินค้านี้ (ไม่บังคับ)

#### ตัวเลือกสินค้า (Variants)

หากสินค้ามีหลายตัวเลือก (เช่น ราคาต่างกัน ประเภทต่างกัน):

1. คลิก **เพิ่มตัวเลือก**
2. กรอก:
   - **ชื่อตัวเลือก** — เช่น "1 เดือน", "3 เดือน", "ภาษาไทย"
   - **ราคาตัวเลือก** — ราคาเฉพาะสำหรับตัวเลือกนี้
   - **ราคา VIP** — ราคา VIP สำหรับตัวเลือกนี้ (ไม่บังคับ)
3. เพิ่มตัวเลือกได้หลายรายการตามต้องการ

3. คลิก **บันทึกสินค้า**
4. สินค้าปรากฏในรายการและบนหน้าร้านทันที

### การแก้ไขสินค้า

1. ค้นหาสินค้าในรายการ
2. คลิกปุ่ม **แก้ไข**
3. อัปเดตช่องที่ต้องการ
4. คลิก **บันทึก**
5. การเปลี่ยนแปลงปรากฏบนหน้าร้านทันที

### การลบสินค้า

1. คลิกปุ่ม **ลบ** ถัดจากสินค้า
2. ยืนยันการลบ

คำเตือน: การลบสินค้าจะลบข้อมูลสินค้าออกจากหน้าร้าน แต่ไม่กระทบออเดอร์ที่มีอยู่แล้ว

### การอัปโหลดรูปภาพสินค้า

รูปภาพสินค้าถูกจัดเก็บบน Cloudflare R2:

1. ในฟอร์มสินค้า คลิกพื้นที่อัปโหลดรูปภาพ
2. เลือกไฟล์รูปภาพจากอุปกรณ์ของคุณ
3. รูปภาพจะอัปโหลดและแสดงตัวอย่าง
4. หากต้องการเปลี่ยนรูป ให้คลิกพื้นที่อัปโหลดอีกครั้งเพื่อเลือกไฟล์ใหม่

## การค้นหาและกรองสินค้า

บนหน้าจัดการสินค้า:

- **ช่องค้นหา** — พิมพ์ชื่อสินค้าเพื่อกรองรายการ
- **ตัวกรองหมวดหมู่** — เลือกหมวดหมู่เพื่อดูเฉพาะสินค้าในหมวดนั้น
- **ตัวกรองประเภท** — กรองตามประเภทสินค้า (ดิจิทัล จัดส่ง บริการ)

ตัวกรองเหล่านี้ช่วยให้ค้นหาและจัดการสินค้าได้รวดเร็วเมื่อร้านมีสินค้าจำนวนมาก
