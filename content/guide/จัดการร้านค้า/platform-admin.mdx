# การดูแลระบบระดับแพลตฟอร์ม

ส่วนการดูแลระบบแพลตฟอร์ม (`/[locale]/portal/platform`) เข้าถึงได้เฉพาะผู้ดูแลระบบแพลตฟอร์ม (superadmin) เท่านั้น ให้การควบคุมระดับสูงสุดเหนือร้านค้าทั้งหมดบนแพลตฟอร์ม

## การเข้าถึง

ต้องมีสิทธิ์ระดับ Platform Admin หากคุณพยายามเข้าถึงโดยไม่มีสิทธิ์ ระบบจะเปลี่ยนเส้นทางคุณออก

---

## การจัดการร้านค้าทั้งหมด

ผู้ดูแลระบบแพลตฟอร์มสามารถดูและจัดการ site ทั้งหมดบนแพลตฟอร์ม:

- ดูรายการ site ทั้งหมด
- เปิด/ปิดใช้งาน site ใดๆ
- เข้าถึง admin dashboard ของ site ใดก็ได้
- ดูสถิติการใช้งานของแต่ละ site

---

## การจัดการผู้ใช้ระดับแพลตฟอร์ม

### การดูผู้ใช้ทั้งหมด

ผู้ดูแลระบบแพลตฟอร์มสามารถดูบัญชีผู้ใช้ทั้งหมดที่ลงทะเบียนในระบบ ครอบคลุมทุก site

### การจัดการบัญชีผู้ใช้

- ค้นหาผู้ใช้ด้วยอีเมล
- ดูข้อมูลบัญชีและกิจกรรม
- ระงับหรือยกเลิกการระงับบัญชี
- รีเซ็ตรหัสผ่าน

---

## การตั้งค่าผู้ให้บริการขนส่ง

ผู้ดูแลระบบแพลตฟอร์มตั้งค่า API credentials ของบริษัทขนส่งที่ site ทั้งหมดจะใช้:

### iShip

1. ไปที่ **Platform → Shipping Carriers**
2. เลือก iShip
3. กรอก:
   - API Key
   - Secret
   - ข้อมูล endpoint
4. บันทึก
5. Site ทั้งหมดที่เปิดใช้งาน iShip จะใช้ credentials เหล่านี้

### Pranakorn Shipping

ตั้งค่าในลักษณะเดียวกันสำหรับผู้ให้บริการขนส่งของแพลตฟอร์มเอง

---

## การจัดการโดเมน

ผู้ดูแลระบบแพลตฟอร์มจัดการการกำหนดค่าโดเมนสำหรับ site ทั้งหมด:

- ดูโดเมนทั้งหมดที่ลงทะเบียน
- ตรวจสอบสถานะ SSL
- เพิ่มหรือลบ custom domain
- ตั้งค่า DNS records

---

## แผนการสมัครสมาชิก

ผู้ดูแลระบบแพลตฟอร์มจัดการแผนการสมัครสมาชิกที่เจ้าของร้านค้าสามารถเลือกได้:

### การดูแผนทั้งหมด

ไปที่ **Platform → Subscription Plans** เพื่อดูรายการแผนที่มีอยู่:
- ชื่อแผน
- ราคา (รายเดือน/รายปี)
- จำนวน site ที่อนุญาต
- ฟีเจอร์ที่รวมอยู่

### การสร้างแผนใหม่

1. คลิก **สร้างแผน**
2. กรอก:
   - ชื่อแผน
   - ราคา
   - จำนวน site สูงสุด
   - ฟีเจอร์ที่รวมอยู่
3. บันทึก

### การมอบหมายแผนให้เจ้าของร้านค้า

1. ค้นหาเจ้าของร้านค้าในระบบ
2. เปิดโปรไฟล์
3. เลือกแผนที่ต้องการมอบหมาย
4. กำหนดวันหมดอายุ (หากมี)
5. บันทึก

---

## ประวัติอีเมลระดับแพลตฟอร์ม

ผู้ดูแลระบบแพลตฟอร์มสามารถดูประวัติอีเมลทั้งหมดจากทุก site บนแพลตฟอร์ม:

- ค้นหาอีเมลตามผู้รับ วันที่ หรือประเภท
- ตรวจสอบปัญหาการส่งอีเมล
- ดูสถิติการส่งอีเมล

---

## การมอนิเตอร์แพลตฟอร์ม

ผู้ดูแลระบบแพลตฟอร์มเข้าถึงตัวชี้วัดระดับระบบ:

- จำนวน site ที่ใช้งานอยู่
- ออเดอร์รวมทั้งแพลตฟอร์ม
- รายได้รวม
- ผู้ใช้ที่ลงทะเบียนใหม่

ข้อมูลเหล่านี้ช่วยให้ผู้ดูแลระบบแพลตฟอร์มตัดสินใจเกี่ยวกับทิศทางและการพัฒนาของแพลตฟอร์มต่อไป
