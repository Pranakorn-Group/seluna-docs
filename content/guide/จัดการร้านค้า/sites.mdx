# การจัดการ Sites (ร้านค้า)

Site คือร้านค้าออนไลน์หนึ่งร้านบนแพลตฟอร์ม เจ้าของร้านหนึ่งคนสามารถมีหลาย Site ได้ขึ้นอยู่กับแผนการสมัครสมาชิก

## การดู Sites ของคุณ

ไปที่ **Portal → Sites** (`/[locale]/portal/sites`) เพื่อดูรายการ site ทั้งหมดของคุณ

แต่ละ site แสดง:
- ชื่อร้านค้า
- โดเมน/ที่อยู่ร้านค้า
- สถานะ (ใช้งานอยู่/ปิดใช้งาน)
- วันที่สร้าง
- ปุ่มจัดการ

---

## การสร้าง Site ใหม่

1. คลิก **สร้างร้านค้าใหม่** หรือ **New Site**
2. ตัวช่วยสร้าง (wizard) จะเปิดขึ้นพร้อมขั้นตอนต่างๆ:

### ขั้นตอนที่ 1: ข้อมูลพื้นฐาน

- **ชื่อร้านค้า** — ชื่อที่แสดงในระบบและบนหน้าร้าน
- **Slug** — ส่วนของ URL เฉพาะสำหรับร้านค้านี้ (เช่น `my-shop` จะเข้าถึงได้ที่ `domain.com/my-shop`)

### ขั้นตอนที่ 2: โดเมน

- กำหนดโดเมนสำหรับร้านค้า:
  - **Subdomain** — ใช้ subdomain ที่ให้มาโดยแพลตฟอร์ม
  - **Custom Domain** — ใช้โดเมนของคุณเอง (ต้องตั้งค่า DNS)

### ขั้นตอนที่ 3: การยืนยัน

- ตรวจสอบข้อมูลทั้งหมด
- คลิก **สร้างร้านค้า**
- Site ใหม่ถูกสร้างและปรากฏในรายการ

### หลังสร้าง Site

หลังจากสร้าง Site แล้ว คุณต้องเข้าไปตั้งค่าเพิ่มเติมใน Admin Dashboard ของ Site นั้น:
- อัปโหลดโลโก้และปรับแต่งธีม
- เพิ่มสินค้าและหมวดหมู่
- ตั้งค่าการชำระเงิน
- ตั้งค่าการจัดส่ง (หากขายสินค้าจริง)

---

## การตั้งค่า Site

คลิกปุ่ม **จัดการ** หรือ **ตั้งค่า** ถัดจาก site เพื่อเข้าถึงการตั้งค่าระดับ site:

### โดเมนและ URL

- เปลี่ยนโดเมนหรือ subdomain
- เพิ่ม custom domain
- ดูสถานะ SSL ของโดเมน

### สมาชิกทีม

- เพิ่มผู้ใช้เป็น admin ของ site
- กำหนดบทบาท:
  - **ADMIN** — เข้าถึง admin dashboard ได้เต็มรูปแบบ
  - **USER** — เข้าถึงได้จำกัด

### สถานะ Site

- **เปิดใช้งาน** — site ทำงานปกติ ลูกค้าเข้าถึงได้
- **ปิดใช้งาน** — site ถูกปิด ลูกค้าไม่สามารถเข้าถึงได้

---

## การลบ Site

**คำเตือน:** การลบ site จะลบข้อมูลทั้งหมดรวมถึงสินค้า ออเดอร์ และบัญชีลูกค้าอย่างถาวร

1. ไปที่การตั้งค่าของ site ที่ต้องการลบ
2. เลื่อนลงไปที่ **Danger Zone**
3. คลิก **ลบ Site**
4. พิมพ์ชื่อ site เพื่อยืนยัน
5. คลิก **ยืนยันการลบ**

ขอแนะนำให้ส่งออกข้อมูลสำคัญก่อนลบ site
